# Neo Santiago 2130 — La ciudad de los rotos

**Novela distópica de Diego Bastías Alfaro**

Lector web de la novela *Neo Santiago 2130: La ciudad de los rotos*, desplegado como sitio estático.

---

## Descripción

Sitio editorial de una sola página para la presentación y lectura de la novela. Incluye portada, sinopsis, índice de capítulos navegable y área de lectura con modo claro/oscuro.

Construido en HTML5, CSS y JavaScript puro. Sin dependencias externas. Sin frameworks.

---

## Estructura del repositorio

```
neo-santiago-2130/
├── index.html       # Sitio completo (HTML + CSS + JS en un solo archivo)
├── README.md        # Este archivo

```

---

## Despliegue en Render

1. Crea un nuevo **Static Site** en [render.com](https://render.com)
2. Conecta este repositorio
3. Configura así:

| Campo | Valor |
|---|---|
| **Build Command** | *(dejar vacío)* |
| **Publish Directory** | `.` |

4. Haz clic en **Create Static Site**

El sitio quedará disponible en una URL del tipo `https://neo-santiago-2130.onrender.com`.

---

## Uso local

Abre `index.html` directamente en cualquier navegador. No requiere servidor.

---

## Características

- Modo oscuro y modo claro
- Barra de progreso de lectura
- Índice navegable con scroll suave
- Drop cap en cada capítulo
- Diseño responsivo (móvil, tablet, escritorio)
- 100% offline — sin llamadas a CDNs ni fuentes externas
